  1. Убедитесь, что сервис каталога запущен на порту 8001
  2. Нажмите "Открыть админ-панель" для перехода к интерфейсу
  3. В админ-панели проверьте:
    • Загрузку списка товаров при открытии страницы
    • Создание нового товара через форму
    • Редактирование существующего товара (кнопка "Редактировать")
    • Удаление товара (кнопка "Удалить" с подтверждением)
  4. Проверьте отображение сообщений об ошибках и успешных операциях
