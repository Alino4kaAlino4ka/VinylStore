# Объединение кнопок покупки и добавления в корзину

## Что было сделано:

### 1. HTML изменения (`src/index.html`)
- Удалены все кнопки `.add-to-cart-btn` ("В корзину")
- Все кнопки заменены на `.buy-btn` с дизайном оранжевой кнопки
- Добавлены `data-product-id` атрибуты для идентификации товаров
- Обновлены цены в тексте кнопок

### 2. JavaScript изменения (`src/scripts/script.js`)
- Добавлены обработчики событий для всех кнопок `.buy-btn`
- Кнопки теперь добавляют товары в корзину при нажатии
- Сохранена вся функциональность добавления в корзину

### 3. CSS изменения (`src/styles/style.css`)
- Добавлены стили для кнопок покупки в коллекции
- Кнопки имеют оранжевый цвет (#FF9900) как основная кнопка
- Добавлены эффекты hover и active
- Кнопки занимают всю ширину карточки товара

## Результат:

✅ **Единый дизайн** - все кнопки имеют одинаковый оранжевый стиль
✅ **Единая функциональность** - все кнопки добавляют товары в корзину
✅ **Сохранена информация** - цены отображаются в тексте кнопок
✅ **Улучшенный UX** - пользователь видит одну понятную кнопку действия
