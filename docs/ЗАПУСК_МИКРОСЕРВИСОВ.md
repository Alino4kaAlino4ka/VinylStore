# Запуск всех микросервисов

Для удобства разработки был создан специальный скрипт `run_all_services.bat`, который позволяет одновременно запустить все три микросервиса ("Каталог", "Корзина", "Заказы") в отдельных окнах терминала.

## Как использовать

1.  Убедитесь, что вы находитесь в корневой директории проекта.
2.  Выполните в терминале следующую команду:

    ```bash
    run_all_services.bat
    ```

## Что делает скрипт

Скрипт выполняет следующие действия:

1.  Запускает **сервис "Каталог"** на порту `8002`.
2.  Запускает **сервис "Корзина"** на порту `8003`.
3.  Запускает **сервис "Заказы"** на порту `8004`.

Все сервисы запускаются с флагом `--reload`, что означает автоматическую перезагрузку при изменении исходного кода. Это избавляет от необходимости вручную перезапускать каждый сервис во время разработки.
